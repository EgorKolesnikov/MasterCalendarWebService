# Описание API

Объект событие:
| Название        | Тип           |
| --------------- |:-------------:|
| ID              | int           |
| Начало          | timestamp     |
| Конец           | timestamp     |
| Описание        | string        |

Ручки:
| Описание          | Тип           |
| ---------------   |:-------------:|
| Добавить событие  | POST          |
| Удалить событие   | DELETE        |
| Изменить событие  | PATCH         |
| Получить событие  | GET           |
